Getting Started

Our REST API is designed to be simple and predictable. It accepts a base64-encoded image and returns the processed image. To get started, you'll need an API key, which is available to all customers on our Enterprise plan.

Authentication

Authenticate your API requests by including your secret API key in the request header. All API requests must be made over HTTPS.

Authorization: Bearer YOUR_API_KEY

Endpoint: /v1/remove

This is the primary endpoint for removing watermarks. It accepts a POST request with a JSON body.

Request Body

  • image_b64 (string, required): The base64-encoded string of the source image.
  • mime_type (string, required): The MIME type of the source image (e.g., "image/jpeg", "image/png").
  • output_format (string, optional): The desired output format. Defaults to "png".

Example Request

Here's an example using JavaScript's fetch API to remove a watermark.


const API_KEY = 'YOUR_API_KEY';
const API_ENDPOINT = 'https://api.watermarkremover.ai/v1/remove';

async function removeWatermark(base64Image, mimeType) {
  const response = await fetch(API_ENDPOINT, {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      image_b64: base64Image,
      mime_type: mimeType
    })
  });

  if (!response.ok) {
    throw new Error(`API Error: ${response.statusText}`);
  }

  const result = await response.json();
  // result.image_b64 contains the processed image
  console.log('Success!', result.image_b64);
  return result.image_b64;
}
